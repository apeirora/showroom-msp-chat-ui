# Security Policy

## About This Project

This repository contains a **demo operator** built for the
[Apeiro](https://github.com/apeirora) architecture. It is intended solely for
demonstration, experimentation, and educational purposes within the ApeiroRA
showroom environment.

## Security Fixes

Because this is a demo project, **no security fixes are guaranteed**. Issues
may be addressed on a best-effort basis, but there is no commitment to timely
patches or ongoing maintenance of security vulnerabilities.

## Reporting a Vulnerability

If you discover a security issue, you are welcome to open a GitHub issue or
contact the maintainers directly. Please be aware that resolution is not
guaranteed and response times may vary.
